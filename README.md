# Kropotkin
Simulation code used to generate results presented in [Meacock & Mitri 2026](https://doi.org/10.32942/X2MD57).

The folder 'Jupyter notebooks' contains python code used to simulate EO systems presented in this manuscript. For more details, please refer to the docs for [EOsimSuite](https://github.com/Pseudomoaner/EOsimSuite) (used to set up and run simulations) and markdown within the workbooks themselves.
